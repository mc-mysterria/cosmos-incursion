# Zone Tiers

Incursion event zones are assigned one of four tiers at event start, distributed according to `zones.tier-distribution` config ratios.

## Tier Summary

| Tier   | Item Drop Chance | Regression | Reward |
|--------|-----------------|------------|--------|
| GREEN  | 0%              | Never      | Low    |
| YELLOW | 33% (configurable) | Never   | Medium |
| RED    | 100%            | Never      | High   |
| DEATH  | 100%            | Seq 4–5 only | Best |

## Tier Mechanics

### GREEN
- Safest option, no item loss, no sequence penalty
- Reward command executed on kill (configurable)
- Boundary particles: green (`00DC00`)

### YELLOW
- 33% chance each item drops on death
- No sequence regression
- Better kill rewards than GREEN
- Boundary particles: yellow (`FFDC00`)

### RED
- All items drop on death
- No sequence regression
- Good kill rewards
- Boundary particles: red (`FF3C3C`)

### DEATH
- All items drop on death
- Sequence regression applies to **Seq 4–5 beyonders only**
- Death penalty logic (see [COMBAT_DEATH.md](COMBAT_DEATH.md))
- Best kill rewards
- Boundary particles: dark red (`500000`)

## Zone Generation

Zones are generated by `ZonePlacerToolkit` with multi-layer validation:
1. Random angle + radius (50–90% of spawn-to-town distance) + ±100 block offset
2. Biome check: rejects ocean, deep ocean, river
3. Surface check: rejects underwater or lava spawn points
4. Area scan: rejects if >30% of 60×60 block area is water
5. Town buffer: rejects locations within configured distance of town claims
6. Separation: enforces minimum distance between zone centers

Total zone count = `max(base-count, online-players / players-per-zone)`, capped at `max-count`.

## Consent System

Players approaching a zone boundary see `ConsentGUI` — a tier-aware agreement screen listing the specific rules for that tier (drop chance, regression risk). Consent is tracked per-player per-event and reset when the event ends. Players already inside a zone when an event starts are teleported outside.

## Player Tier Classification

Inside any zone, players are classified:

| PlayerTier     | Criteria          | Effects                          |
|----------------|-------------------|----------------------------------|
| SPIRIT_WEIGHT  | Beyonder Seq 4–5  | Mandatory glow + DOT damage      |
| INSIGNIFICANT  | Seq 6–9 or normal | No forced effects, stealth viable |

SPIRIT_WEIGHT players are also marked on the BlueMap.
